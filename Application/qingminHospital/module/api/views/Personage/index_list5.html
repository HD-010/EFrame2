学习贯彻十八大精神 推进医院科学发展

发布时间：2012/12/29    浏览：19465次    来源：本站原创

 

 

 

 

 

      为进一步深入学习宣传贯彻党的十八大精神，把党员干部职工的思想和行动统一到党的十八大精神上来，推进医院科学发展，努力为人民群众提供“安全、有效、方便、价廉”的医疗服务。12月28日，医院学习贯彻党十八大精神专题辅导报告会在新门诊楼四楼会议厅举行。党委书记王义文作辅导报告，党委委员、副院长张平主持会议，院党委委员及全体党员参加了会议。

      王书记的辅导报告从党代会知识、十八大概况、时代背景(世情、国情、党情)、主要精神(十二个方面)和学习宣传贯彻十八大精神等五个方面对十八大报告进行了全面、深入的辅导。王书记在报告中强调，要把学习好、宣传好、贯彻好好党的十八大精神作为医院当前和今后一个时期的首要政治任务。医院党组织、党员要增强学习十八大精神的主动性与自觉性，深刻领会十八大报告精神实质；要贯彻落实“党要管党、从严治党”的要求。通过学习十八大精神，进一步抓好党组织的思想建设、组织建设、作风建设、能力建设、反腐倡廉建设、先进性和纯洁性建设，建设学习型、服务型、创新型医院党组织；要把学习活动与业务工作紧密结合，推进医院各项事业的发展，不断提升服务质量和服务水平，努力为群众提供“安全、有效、方便、价廉”的医疗服务。

      张平副院长在会议总结时指出，学习好、宣传好、贯彻好十八大精神，是动员和激励党员干部职工解放思想、实事求是、与时俱进、求真务实，以更加奋发有为的精神状态投身医院改革发展实践的迫切需要；是进一步把科学发展观贯彻到医院改革发展全过程、体现到党的建设各方面的迫切需要。即将到来的2013年，是全面贯彻落实党的十八大精神的开局之年和实施医院“十二五”规划承前启后的重要一年。王书记的辅导报告内容丰富，深入浅出，结合实际，我们要认真领会，落实行动。始终保持党的纯洁性和先进性，立足本职，创先争优，使医院党组织更富创造力、凝聚力和战斗力，党员先锋模范作用进一步体现和发挥，为推进医院又好又快发展贡献我们的智慧和力量。
