医院简介
当前您所在的位置：首页 > 医院概况 > 医院简介


      宣城市人民医院是集医教研和预防保健等于一体的三级甲等综合医院。是省辖宣城市最大的公立医院，皖南医学院、蚌埠医学院教学医院。曾获全国文明单位、国家爱婴医院、省级诚信医院、省级医德医风示范医院、省先进基层党组织、省卫生系统先进集体、省三八红旗集体、省卫生先进单位、市民主考评先进单位、市人才工作先进集体等称号。

      现开放床位1100张，临床科室28个，病区26个，医技科室9个；年门诊60余万人次、出院4万余人次、住院手术近万台次。在岗职工1200余人，其中高级职称112人（正高26人）、教授（副教授）15人、博士6人、硕士108人、中级职称近400人，多人入选省先进工作者、省杰出专业技术人才、市优秀人才、市拔尖人才、市十佳科技工作者等。

      医院承担着宣城市辖区（五县一市一区）及周边近300万人民群众的防病治病任务。省重点培育学科1个（心血管内科），市重点学科10个，院重点学科14个、特色专科2个。三甲医院专科技术齐备，常规开展体外循环下心脏外科手术、冠脉搭桥术、心脏起搏术、射频消融术、冠脉支架植入术、先心介入治疗术、自体干细胞移植术、颈胸腰椎前(后)入路手术、后腹腔镜下肾脏及肾上腺手术、ERCP、ESD、腔镜下无金属残留胆囊切除术、肝|胰十二指肠切除术、宫颈癌根治术、喉癌根治术等先进技术。同时拥有全市领先水平的超声诊断科、病理科、影像科、检验科、输血科、电生理科、临床药学科等。现有先进诊疗装备千余台套，影像装备处全省先进水平。

      作为公立医院，医院始终突出公立医院公益性，坚定保障病人利益。药耗占比、次均费用全省同级医院最低，可报销比、实际补偿比处全市前列。

      医院始终秉承“病人利益至上”的服务宗旨和“博爱、仁和、诚信、创新”的医院精神，坚持“医教研并重、质量兴院、服务兴院、人才强院、科教兴院”发展道路，励精图治，开拓创新，全力打造政府放心、群众满意、职工爱戴的现代化区域医学中心。
